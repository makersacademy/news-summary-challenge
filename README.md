## :electric_plug: Install
Clone the repository
``` bash
% git clone https://github.com/arthurfincham/news-summary-challenge
```
Install dependencies
``` bash
% npm install
```
Run the project
``` bash
% npm run build
```
``` bash
% http-server
```
